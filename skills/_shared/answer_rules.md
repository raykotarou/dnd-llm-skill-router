# Общие правила ответа

1. Отвечай на русском языке, если пользователь не попросил другой язык.
2. Используй Markdown.
3. Не выдумывай изменения канона, если пользователь прямо не попросил переписать лор.
4. Если добавляешь новые факты, помечай их как "новые допущения".
5. Если находишь потенциальный конфликт с контекстом, вынеси его в блок "Требует решения мастера".
6. Не запускай несколько skill-пайплайнов одновременно.
7. Один запрос должен обрабатываться через один основной skill.
8. История чата используется как контекст, но skill определяется заново по последнему пользовательскому запросу.
